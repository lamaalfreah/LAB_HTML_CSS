KhutbaTech

KhutbaTech is my graduation project: a real-time AI-powered platform that transcribes and translates Friday sermons from Al-Masjid Al-Nabawi. The system uses FastAPI and Azure Speech-to-Text for backend processing, and integrates GPT-4o for accurate translation while preserving Quranic content. The front-end was developed using HTML, CSS, and JavaScript, featuring a responsive interface that allows users to view the Arabic transcript, English translation, and download past sermons.
